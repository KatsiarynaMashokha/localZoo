# Local Zoo
#### Local Zoo is written in JavaScript with the help of the following tools: HTML, CSS, Bootstrap, jQuery, Node, Angular. Published on 09/22/2017.
#### By **Katsiaryna Mashokha**
## Description
Local Zoo is an app to keep track of animals in a Zoo.

## Development Specifications
| Behavior      | Example Input         | Example Output        |
| ------------- | ------------- | ------------- |
| App can display list of animals in the zoo | Homepage | ('Arctic Fox', 'Moon', 2, 'Carnivore', 'Northern Trail', 5, 'Female', 'Cool shade', 'Loud noises')|
| A new animal can be added to the Zoo |  'Ocelot', 'Prince', 4, 'Carnivore', 'Tropical Rain Forest Building', 6, 'Male', 'Laying in the sunshine', 'Toys that are not rope-based'  | 'Ocelot', 'Prince', 4, 'Carnivore', 'Tropical Rain Forest Building', 6, 'Male', 'Laying in the sunshine', 'Toys that are not rope-based'|
| App can execute a search base on the animal's age |  Maximum age = 2  |('Arctic Fox', 'Moon', 2, 'Carnivore', 'Northern Trail', 5, 'Female', 'Cool shade', 'Loud noises') |
| Animal can be edited| Artic Fox Moon: new age = 3| ('Arctic Fox', 'Moon', 3, 'Carnivore', 'Northern Trail', 5, 'Female', 'Cool shade', 'Loud noises')|
